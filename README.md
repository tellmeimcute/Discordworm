# Discordworm 
Программа заставляющая приложение discord под windows использовать SOCKS5-прокси. Умеет проксировать TCP и UDP трафик. 
Опционально можно отключить проксирование UDP трафика и вместо этого программа будет отправлять фейк на discord_ip_discovery.
Содержимое фейка можно изменить в конфиге.

## Зачем это существует?
Я не нашел решения, которое позволяет проксировать TCP и предлагает выбор между проксироваем UDP трафика и отсылкой фейка.

### Использование
1. Закинуть DWrite.dll в папку %localappdata%\Discord\{ПОСЛЕДНЯЯ_ВЕРСИЯ}\
2. Создать в этой же папке dwormconf.txt.
3. Настроить конфиг:
```
proxy_address=PROXY_IPV4
proxy_port=PROXY_PORT
proxy_udp=1|0
fake_udp_payload=0xHEX
```

**proxy_udp** - Включить (1) проксирование всего UDP трафика. Выключить (0). 
Если выключено - будет посылать fake_udp_payload на каждый discord ip discovery.

**fake_udp_payload** - HEX строка максимум 256 байт. Если не указано будет 256 нулей.
